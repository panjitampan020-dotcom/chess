# Chess Simulator

## Documentation

The Chess Simulator is a program that allows users to simulate chess games. It supports move generation, validation, and user interaction via a command line interface.

### Features
- Play against AI or another player.
- Move validation and illegal move handling.
- History of moves and easy undo functionality.

## Setup Instructions

### Requirements
- Python 3.8 or higher
- Required libraries (install via `pip`):
  - numpy
  - chess

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/panjitampan020-dotcom/chess.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chess
   ```
3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Simulator
To start the chess simulator, run:
```bash
python main.py
```
